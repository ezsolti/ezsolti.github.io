Reactor physics¶

  1. Neutron transport in nuclear reactors
    • Lot of data to be handled (cross sections)
    • Equations to be solved numerically
    • Simulations with Monte Carlo methods
    • Summa: computers are essential
  2. Energy Physics II with Nuclear Energy
    • MSc level, 10c, Fission+Fusion
    • JUNE
      • Introduce Jupyter Home assignments/Jupyter based lecture notes
    • To bring computations into the curriculum
  3. Ended up creating a standalone "Reactor physics with Python" course
    • Life long learning summer distance course

Outcome¶

  • Reactor physics with Python
    • Intro to reactor physics/data analysis/scientific computing
    • Distance course: students work alone on datalabs
    • Very active learning
  • 11 datalabs (1-2 jupyter notebooks/lab)
    • Computer "experiments"
    • Instruction and code skeleton to complete
    • Prepared code and parameters to change
  • 8 home assignments: Instructions and inputs
  • Lecture notes based on our own figures
    • 70+ figures with source code supplemented

Datalab syllabus (datalab = jupyter notebook)¶

  1. Introduction to python (dict, list, for, while, if, else, I/O)
  2. Intermediate python I (numpy, matplotlib, scipy, regexp)
  3. Intermediate python II (Class, JSON, xml, csv, pandas)
  4. Neutron scattering and Monte Carlo methods (random numbers, pdf sampling
  5. Fission trees and fixed source Monte Carlo
  6. Introduction to openMC (free MC code developed by MIT)
  7. Numerical solutions to the 1G-1D diffusion equation and group constant generation
  8. Point kinetics with prompt, and delayed neutrons (1group/6groups)
  9. Experimental reactor physics: subcritical measurements
  10. Scripting openMC: void coefficient
  11. Depletion calculations with openMC

Batch-wise computation is needed

    keffs = []
    for all batches:
        Nnew = 0
        for all neutrons in batch:
            if neutron enters fission:
                increment Nnew with the number of neutrons emerging from fission
        k-effective estimate of batch = Nnew/(neutrons per batch)
        append k-effective estimate to the list keffs
    final k-effective estimate is mean(keffs)
