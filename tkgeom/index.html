tkgeom: An example python package¶

tkgeom is a python package, intended to serve as an example for the TK workshop

As a package, tkgeom provides

  • basic 2D geometry classes (Point, Segment, Circle, Rectangle)

The package can be found in a github repo. Below you can find an example notebook describing how to interact with tkgeom.
