Rings: 0 (S=1)
Right-click a cell to change its universe.
Affects Serpent TYPE and export.
Used only for Serpent lat header (X0=Y0=0).
(0 is allocated for surrounding/empty).

Instructions

  • Pan: Hold and drag. Arrow keys also pan.
  • Zoom: Mouse wheel. Hold Shift + wheel for horizontal pan, Alt + wheel for vertical pan.
  • Fill (single-click): Check Fill, pick a universe, then left-click a cell.
  • Brush paint (drag): Check Fill, then hold Shift or Ctrl while dragging.
  • Symmetry: Choose ×3 or ×6 to paint rotationally symmetric sets.
  • Rect fill: Check Rect fill, drag to select. The shaded rectangle shows the area to be filled.
  • Context menu: Right-click a cell to assign its universe.
  • Add ring: Choose a universe, click Add ring to add the next outer ring.
  • Reassign last ring: Pick a universe, click Apply to change only the outermost ring.
  • Undo: Removes the last added ring (preserving extended edits, if any).
  • Show extended: Toggle to view & edit the full N×N parallelogram (outside the hex disk).
  • Export: Print / Save lattice.txt emits the Serpent table:
    • TYPE=2 (X-type, pointy-top): row-major (bottom→top, left→right)
    • TYPE=3 (Y-type, flat-top): column-major (printed as transpose)
