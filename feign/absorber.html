blocks.Absorber()¶

class feign.blocks.Absorber(absID=None)¶

A class used to represent an Absorber. An absorber can be thought of any element around (or within) the assembly, which attenuates gamma radiation.

Parameters

absID (str) – ID of the absorber

ID of the absorber

the shape of the absorber

matID of the Material the absorber is made of

Type

str

accommat¶

matID of the Material the absorber is surrounded with (Note: the program has no capabilities to decide which material is around the absorber, thus the user has to set this)
