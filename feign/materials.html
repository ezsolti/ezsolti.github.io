feign.geometry.Materials()¶

class feign.blocks.Materials(*argv)¶

A class used to represent a Materials container.

It can be thought as a dictionary of Material() objects.

Parameters

*argv (Material()) – One or more Material instances

materials¶

dictionary to store Material objects. Keys are matIDs, values are Material().

Type

dictionary

Examples

>>> uox=Material('1')
>>> zr=Material('2')
>>> materials=Materials(uox,zr)
>>> materials.materials
{'1': Material(matID=1), '2': Material(matID=2)}
add(material)¶

The function to add a Material() object to a Materials() object.

Parameters

material (Material()) – Material to be added

Examples

>>> uox=Material('1')
>>> zr=Material('2')
>>> materials=Materials()
>>> materials.materials
{}
>>> materials.add(uox)
>>> materials.add(zr)
>>> materials.materials
{'1': Material(matID=1), '2': Material(matID=2)}
remove(material)¶

The function to remove a Material() object from a Materials() object. (Note: the Material object is not deleted from memory)

Parameters

material (Material()) – Material to be removed

Examples

>>> uox=Material('1')
>>> zr=Material('2')
>>> materials=Materials([fuel,clad])
>>> materials.materials
{'1': Material(matID=1), '2': Material(matID=2)}
>>> materials.remove(zr)
>>> materials.materials
{'1': Material(matID=1)}
>>> materials.remove(zr)
You can remove only existing Material()
>>> a=Point(3,4)
>>> materials.remove(a)
You can remove only Material()
