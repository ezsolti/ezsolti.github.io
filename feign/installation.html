Installation¶

feign can be installed by downloading the zipball from github.

pip install https://github.com/ezsolti/feign/zipball/master

The main dependencies are:

  • NumPy

  • Matplotlib

  • re

Getting data¶

Total attenuation coefficients can be obtained from XCOM. Test data is pre-prepared for some common materials in passive gamma spectroscopy (uranium-dioxid, water, air, lead, aluminium, copper, stainless steel).

https://github.com/ezsolti/feign/dataFin/zipball/master
