feign.geometry.Absorbers()¶

class feign.blocks.Absorbers(*argv)¶

A class used to represent an Absorbers container.

It can be thought as a dictionary of Absorber() objects.

Parameters

*argv (Absorber()) – One or more Absorber instances

absorbers¶

dictionary to store Absorber objects. Keys are absIDs, values are Absorber().

Type

dictionary

Examples

>>> leadsheet=Absorber('leadsheet')
>>> alusheet=Absorber('alusheet')
>>> absorbers=Absorbers(leadsheet,alusheet)
>>> absorbers.absorbers
{'leadsheet': Absorber(absID=leadsheet), 'alusheet': Absorber(absID=alusheet)}
add(absorber)¶

The function to add an Absorber() object to an Absorbers() object.

Parameters

absorber (Absorber()) – Absorber to be added

Examples

>>> leadsheet=Absorber('leadsheet')
>>> alusheet=Absorber('alusheet')
>>> absorbers=Absorbers()
>>> absorbers.absorbers
{}
>>> absorbers.add(leadsheet)
>>> absorbers.add(alusheet)
>>> absorbers.absorbers
{'leadsheet': Absorber(absID=leadsheet), 'alusheet': Absorber(absID=alusheet)}
remove(absorber)¶

The function to remove an Absorber() object from an Absorbers() object. (Note: the Absorber object is not deleted from memory)

Parameters

absorber (Absorber()) – Absorber to be removed

Examples

>>> leadsheet=Absorber('leadsheet')
>>> alusheet=Absorber('alusheet')
>>> absorbers=Absorbers(leadsheet,alusheet)
>>> absorbers.absorbers
{'leadsheet': Absorber(absID=leadsheet), 'alusheet': Absorber(absID=alusheet)}
>>> absorbers.remove(alusheet)
>>> absorbers.absorbers

>>> absorbers.remove(alusheet)
