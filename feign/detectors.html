feign.geometry.Detectors()¶

class feign.blocks.Detectors(*argv)¶

A class used to represent a Detectors container.

It can be thought as a dictionary of Detector() objects.

Parameters

*argv (Detector()) – One or more Detector instances

detectors¶

dictionary to store Detector objects. Keys are detIDs, values are Detector().

Type

dictionary

Examples

>>> F5=Detector('F5')
>>> F15=Detector('F15')
>>> detectors=Detectors(F5,F15)
>>> detectors.detectors
{'F5': Detector(detID=F5), 'F15': Detector(detID=F15)}
add(detector)¶

The function to add a Detector() object to a Detectors() object.

Parameters

detector (Detector()) – Detector to be added

Examples

>>> F5=Detector('F5')
>>> F15=Detector('F15')
>>> detectors=Detectors()
>>> detectors.detectors
{}
>>> detectors.add(F5)
>>> detectors.add(F15)
>>> detectors.detectors
{'F5': Detector(detID=F5), 'F15': Detector(detID=F15)}
remove(detector)¶

The function to remove a Detector() object from a Detectors() object. (Note: the Detector object is not deleted from memory)

Parameters

detector (Detector()) – Detector to be removed

Examples

>>> F5=Detector('F5')
>>> F15=Detector('F15')
>>> detectors=Detectors(F5,F15)
>>> detectors.detectors
{'F5': Detector(detID=F5), 'F15': Detector(detID=F15)}
>>> detectors.remove(F15)
>>> detectors.detectors
{'F5': Detector(detID=F5)}
>>> detectors.remove(F15)
You can remove only existing Detector()
