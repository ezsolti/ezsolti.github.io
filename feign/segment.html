intersection(other)¶

The function to find the intersection of two Segment objects.

Parameters

other (Segment()) – The Segment() for which the intersection is calculated

Returns

list of the intersections

Return type

list of Point()

Notes

The list has one element if the Segments intersect (even if only through their end points).

An empty list is returned if the Segments do not have an intersection.

Examples

>>> s1=Segment(Point(2,2),Point(-2,-2))
>>> s2=Segment(Point(-2,2),Point(2,-2))
>>> s1.intersection(s2)
 [Point(0.000, 0.000)]
>>> s3=Segment(Point(-3,-4),Point(-6,-7))
>>> s1.intersection(s3)
 []
