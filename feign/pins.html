feign.geometry.Pins()¶

class feign.blocks.Pins(*argv)¶

A class used to represent a Pins container.

It can be thought as a dictionary of Pin() objects.

Parameters

*argv (Pin()) – One or more Pin instances

pins¶

dictionary to store Pin objects. Keys are pinIDs, values are Pin().

Type

dictionary

Examples

>>> fuel=Pin('1')
>>> guide=Pin('2')
>>> pins=Pins(fuel,guide)
>>> pins.pins
{'1': Pin(pinID=1), '2': Pin(pinID=2)}
add(pin)¶

The function to add a Pin() object to a Pins() object.

Parameters

pin (Pin()) – Pin to be added

Examples

>>> fuel=Pin('1')
>>> guide=Pin('2')
>>> pins=Pins()
>>> pins.pins
{}
>>> pins.add(fuel)
>>> pins.add(guide)
>>> pins.pins
{'1': Pin(pinID=1), '2': Pin(pinID=2)}
remove(pin)¶

The function to remove a Pin() object from a Pins() object. (Note: the Pin object is not deleted from memory)

Parameters

pin (Pin()) – Pin to be removed

Examples

>>> fuel=Pin('1')
>>> guide=Pin('2')
>>> pins=Pins(fuel,guide)
>>> pins.pins
{'1': Pin(pinID=1), '2': Pin(pinID=2)}
>>> pins.remove(guide)
>>> pins.pins
{'1': Pin(pinID=1)}
>>> pins.remove(guide)
You can remove only existing Pin()
